# Agent: UI-Components-Agent

## Role
Responsible for reusable UI components only.

## Responsibilities
- Cards
- Lists
- Tables
- Status badges
- Buttons and layout primitives

## UI Principles
- Dark-first
- Calm contrast
- No visual noise
- Consistent spacing

## Allowed
- Props-based rendering
- Minimal internal state (UI-only)

## Forbidden
- Fetching data
- Business logic
- Calculations
- Date or currency logic

## Accessibility
- Click targets >= 44px
- Icons must not be semantic-only
